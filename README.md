# Phương Pháp Tính Giá - Quốc Long — v2 FINAL

Đã sửa đúng theo yêu cầu cuối:
- Tiêu đề: **Phương Pháp Tính Giá - Quốc Long**
- Chi phí quản lý = **% doanh thu FULL VAT**
- Lãi vay tháng = **Vốn vay × Lãi suất năm / 12**
- Lãi vay là **chi phí chung cho cả lô**
- Chỉ tiêu `/chiếc` dùng **lãi vay phân bổ = lãi vay tháng / số lượng**
- Chỉ tiêu `tổng` dùng **1 lần lãi vay tháng**, không nhân theo qty
